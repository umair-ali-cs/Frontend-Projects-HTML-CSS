# Frontend Projects (HTML & CSS)

A collection of frontend clone projects built with HTML, CSS, and JavaScript, focused on replicating real-world UI/UX, layout precision, and responsive design.

## 🎬 Netflix Clone
A pixel-accurate recreation of the Netflix landing page — hero section, subscription CTA, FAQ accordion, and responsive layout.

🔗 **Live Demo:** https://umair-ali-cs.github.io/Frontend-Projects-HTML-CSS/Netflix%20Clone/

## 🔍 Google Clone
A clone of the Google homepage, recreating its minimalist search interface and layout.

🔗 **Live Demo:** https://umair-ali-cs.github.io/Frontend-Projects-HTML-CSS/Google%20Clone/

## 🎾 Blanca Padel Clone
A clone of a padel/sports brand website, focused on layout structure and visual design.

🔗 **Live Demo:** https://umair-ali-cs.github.io/Frontend-Projects-HTML-CSS/Blanca%20Padel%20Clone/
